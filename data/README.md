## Dataset Overview

This folder contains the dataset used for the stroke risk analysis project.

### File
- `stroke.csv`

### Description
The dataset includes demographic and clinical variables commonly associated with stroke risk. It is structured tabular data suitable for statistical analysis and machine learning classification tasks.

Key variables include:
- Age
- Gender
- Hypertension status
- Heart disease
- Average glucose level
- Body mass index (BMI)
- Smoking status
- Stroke occurrence (binary outcome)

### Notes
- The dataset is publicly available and included here to support reproducibility.
- No personally identifiable information is present.
- The raw dataset is intentionally left unchanged.
- All preprocessing, filtering, and feature transformation steps are performed within the analysis script.

This dataset supports exploratory analysis, inferential statistics, and predictive modeling in a healthcare analytics context.
